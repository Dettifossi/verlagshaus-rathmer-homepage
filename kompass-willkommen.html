App einrichten – einmalig, dann dauerhaft

Speichern Sie die App als Icon auf Ihrem Homebildschirm – dann bleiben Sie dauerhaft angemeldet und müssen den Code nie wieder eingeben.

iPhone · Safari
  1. Öffnen Sie Safari und gehen Sie auf:
    kompass.verlagshausrathmer.com
  2. Geben Sie Ihren Zugangscode ein
  3. Tippen Sie unten auf das Teilen-Symbol (Quadrat mit Pfeil nach oben)
  4. Wählen Sie „Zum Home-Bildschirm" → „Hinzufügen"
Android · Chrome
  1. Öffnen Sie Chrome und gehen Sie auf:
    kompass.verlagshausrathmer.com
  2. Geben Sie Ihren Zugangscode ein
  3. Tippen Sie oben rechts auf die drei Punkte (⋮)
  4. Wählen Sie „Zum Startbildschirm hinzufügen"
